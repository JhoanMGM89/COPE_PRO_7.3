1. Descripción General del Sistema

El Generador de Plantillas Pro es una aplicación web diseñada para la gestión operativa de telecomunicaciones. Permite a los agentes generar, copiar y gestionar plantillas estandarizadas para distintas plataformas de trabajo: Averías, ATP/ATC, Service Now y Creación.

El sistema opera como una aplicación de página única (SPA) construida con HTML/CSS/JavaScript puro, empaquetada dentro de un contenedor React + Vite. Cada módulo funciona como un archivo HTML independiente cargado dinámicamente mediante iframes.

Características Principales:

  • Generación automática de plantillas con formato estandarizado
  • Base de datos interna de IPs con más de 200 registros OLT
  • Sistema de historial con exportación a Excel (CSV)
  • Contadores de gestión por categoría
  • Herramienta PUTTY integrada para comandos Nokia/Huawei
  • Personalización visual (fondo e identidad de agente)
  • Persistencia de datos vía localStorage

3. Módulo Central — Generador de Plantillas

HUB GENERADOR_DE_PLANTILLAS.html

Archivo principal (~7200 líneas) que actúa como hub central del sistema. Contiene:

Pantalla de Login

  • Campo de texto para nombre de agente
  • Botón "Cargar imagen de fondo" con soporte para imágenes personalizadas
  • Botón de eliminar fondo (🗑️)
  • Almacenamiento en localStorage (nombreAgente, fondoPersonalizado)

Pantalla de Bienvenida

  • Mensaje "BIENVENIDO, [AGENTE]"
  • Grid 2x2 con botones: AVERÍAS, ATP, CREACIÓN, SERVICE NOW
  • Botón "⚙️ Cambiar agente / fondo" para regresar al login
  • Los botones se ocultan al abrir un módulo y reaparecen al cerrarlo

Módulo CREACIÓN (Modal interno)

A diferencia de los otros módulos, CREACIÓN se ejecuta como un modal dentro del propio archivo, no como iframe externo.

  • Base de datos interna de IPs (~200+ registros OLT)
  • Detección automática de datos: OLT, localidad, coinversión, tecnología, grupo de trabajo, artículo
  • Soporte para múltiples coinversiones: ONNET, ATP, ATC, ONNET MAXIMO, APAGADA
  • Switches de plataforma: SYTEX, GDM, CAZAMASIVOS
  • Switch de múltiples IPs
  • Estados de creación: Nuevo Caso, Anexo
  • Generación de plantillas en 3 formatos: RESUMEN, DETALLE, PLANTILLA COMPLETA
  • Sistema de historial con búsqueda, contadores y exportación
  • Modal de rechazos con formulario específico y vista previa
  • Herramienta PUTTY accesible desde la interfaz

7. Módulo CREACIÓN (Detalle)

CREACIÓN Integrado en GENERADOR_DE_PLANTILLAS.html

Módulo más complejo del sistema. Gestiona la creación de casos de fibra óptica con detección automática de infraestructura.

Flujo de Trabajo

  1. El agente ingresa una IP principal (+ tarjeta/puerto opcionales)
  2. El sistema busca en la base de datos interna y auto-rellena: OLT, Localidad, Coinversión, Tecnología, Grupo de Trabajo, Artículo
  3. Se selecciona el tipo de falla y estado de creación
  4. Se completan datos del caso: FO, incidencia, clientes, observaciones
  5. El sistema genera 3 vistas: Resumen, Detalle y Plantilla Completa

Sistema de Historial

Cada módulo mantiene su propio historial con las siguientes capacidades:

  • Registro automático al copiar plantilla (timestamp, tipo, contenido)
  • Búsqueda/filtrado en tiempo real
  • Vista previa de plantilla (👁️)
  • Copiado directo desde historial (📋)
  • Exportación a Excel (formato CSV con BOM UTF-8)
  • Limpieza total del historial

Sistema de Comunicación (postMessage)

  • 'cerrarModulo' — Cierra el módulo activo y retorna a bienvenida
  • 'cerrarPutty' — Cierra solo el overlay de PUTTY sin afectar al padre
  • {type: 'openModule'} — Abre un módulo con parámetros
